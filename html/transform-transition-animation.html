过渡

过渡效果需在指定属性值变化时才会触发

原始
transition: width 2s, height 2s, background-color 2s
transition: width 2s ease-out, height 2s linear, background-color 2s ease-in
transition: width 2s, height 2s, background-color 2s;transition-delay: 3s;
transition: width 2s, height 2s, background-color 2s, transform 2s;
测试过渡属性是否影响元素占用的位置
transition: all 2s linear 3s;

动画

动画效果会根据动画设置自动触发

原始
animation: one 2s;

一个动画的完整示例，每个属性支持哪些值参考文档

测试动画属性是否影响元素占用的位置

转换和动画都不会影响元素占用的页面位置和区域，过渡由于属性值产生了变化，自然会影响到元素占用的位置和区域
